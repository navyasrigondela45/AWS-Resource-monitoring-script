# Resource Monitoring Script 🚀

## 📌 Overview
This project is part of my DevOps learning journey.  
It focuses on automating system resource monitoring using Linux and Bash scripting.

The script collects system information and generates a report, which can be scheduled and shared via email.

---

## 🔧 Features
- Monitors CPU usage  
- Checks memory usage  
- Tracks disk utilization  
- Generates a system report  
- Automates execution using cron jobs  
- Sends email alerts with the report  

---

## 🛠️ Tech Stack
- Linux  
- Bash Scripting  
- Cron Jobs  
- System Monitoring Commands
